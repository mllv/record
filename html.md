## HTML - 블록 레벨(Block level) 요소와 인라인(Inline level) 요소

1. 블록 요소
	1. DIV, H1, P
	2. 사용 가능한 최대 가로 너비를 사용한.
	3. 크기를 지정할 수 있다.
	4. (width: 100%; height: 0;로 시작)
	5. 수직으로 쌓임
	6. margin, padding 위, 아래, 좌, 우 사용 가능하다.
	7. 레이아웃을 잡는 용도
2. 인라인 요소
	1. SPAN, IMG
	2. 필요한 만큼의 너비를 사용한다.
	3. 크기를 지정할 수 없다.
	4. (width: 0; height: 0;로 시작)
	5. 수평으로 쌓임
	6. margin, padding 위, 아래는 사용할 수 없다.
	7. text를 다루는 용도
